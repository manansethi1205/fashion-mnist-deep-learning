# Project Evolution

This project was developed iteratively rather than using a single model from the beginning.

The progression was:

1. Baseline MLP
2. Hyperparameter optimization using Optuna
3. CNN architecture
4. Identification of overfitting
5. Data augmentation
6. Learning-rate scheduling
7. Final CNN evaluation

The goal was not only to maximize test accuracy, but also to understand how architectural changes and regularization techniques affect model generalization.
